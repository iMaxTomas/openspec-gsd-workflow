# Roadmap

## Phase 1: Windows-local VTM baseline

Goal: stand up the first usable Windows-local terminal workspace for `172.29.111.218`.

### Focus

- Define and validate the preferred VTM-first Windows workflow.
- Lock shell startup to PowerShell 7 (`pwsh`).
- Apply Maple Mono as the baseline font.
- Establish the bounded personalization surface.
- Implement and verify a manual synchronized light/dark toggle for shell plus terminal.
- Keep WezTerm visible as a non-primary parity path.

### Exit criteria

- VTM is the clear primary Windows-local path.
- `pwsh` is the inspectable shell entry surface.
- Maple Mono baseline is confirmed.
- Personalization stays inside the defined workspace boundary.
- Manual light and dark mode switching keeps shell and terminal synchronized.
- WezTerm relevance is recorded without widening scope.
